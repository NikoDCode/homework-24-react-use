# React App с хуком use()

Простое приложение, демонстрирующее новый хук `use()` из React 19 для загрузки списка пользователей.

## Установка и запуск

```bash
npm install
npm run dev
```

## Что используется

- React 19 с хуком `use()`
- TypeScript
- Vite
- Suspense для загрузки
- API: jsonplaceholder.typicode.com
